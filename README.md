# Sales-Data-Analysis-Excel-Dashboard
Sales Data Analysis Dashboard
Project Overview

This project analyzes sales transaction data to identify revenue trends, customer purchasing behavior, and product performance using Microsoft Excel.

Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Excel Formulas
Data Visualization

Business Questions
What is the total sales revenue?
Which product category performs best?
Which customer group contributes the most revenue?
How do sales trends change over time?

Analysis Performed
✔ Sales by Product Category
✔ Sales by Gender
✔ Sales by Age Group
✔ Monthly Sales Trend
✔ KPI Dashboard Creation

Key Insights
Total revenue reached $456,000.
Electronics generated the highest revenue.
Female customers contributed slightly higher sales.
The 18–27 age group generated the highest revenue.
